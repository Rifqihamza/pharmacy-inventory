# Pharmacy Inventory Management System

A lightweight C# console-based inventory system to manage pharmacy medicines: add, update, delete, search, and filter by category.

---

## ğŸ“¦ Features overview

| Feature            | Description                                                               |
| ------------------ | ------------------------------------------------------------------------- |
| Add Medicine       | Insert a new medicine with name, description, category, price, and stock. |
| View All Medicines | Display complete list of medicines with basic details.                    |
| Update Medicine    | Modify an existing medicine by ID.                                        |
| Delete Medicine    | Remove a medicine from the inventory.                                     |
| Search             | Search medicine by name or keyword.                                       |
| Filter             | Filter medicines by category.                                             |
| Auto Increment ID  | IDs are automatically generated by the system.                            |
